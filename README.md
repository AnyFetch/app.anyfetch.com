Anyfetch front
==============

Welcome @ anyFetch

## Requirements

* node.js
* compass
* [yeoman](http://yeoman.io/)


## Installation
Setup npm and bower:

```
npm install
```

## How to use


```sh
grunt serve				# launch the developement server
grunt test				# start the tests
grunt build				# create standalone application in the /build directory
```

## Deployment
 
```sh
grunt deploy
```
 
This require push access to this repo.
